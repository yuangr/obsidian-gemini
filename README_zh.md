# Gemini Scribe for Obsidian (中文说明)

Gemini Scribe 是一款将 Google Gemini AI 模型无缝集成到 Obsidian 中的插件。它直接在 Obsidian 内为您提供强大、AI 驱动的写作、笔记记录以及知识库管理协助。它将您的本地笔记作为上下文提供给 AI，让您可以享受高度定制化且集成良好的 AI 体验。

[English README](README.md)

> **注意：** 在插件设置的 **Provider (提供商)** 中选择以下两种配置路径之一：
>
> - **Google Gemini (云端)** — 需要 Gemini API 密钥（可在 [Google AI Studio](https://aistudio.google.com/apikey) 获取免费层密钥）。
> - **Ollama (本地)** — 本地运行，无需 API 密钥。安装 [Ollama](https://ollama.com)，拉取模型，并在设置中选择它。具体功能差异见 [docs/guide/ollama-setup.md](docs/guide/ollama-setup.md)。

## v4.9.2 新增特性

**⚡ 性能与成本优化**

- **⚡ 并发工具执行** - 独立的只读与搜索类工具支持并发并行执行，缩短 Agent 运行延迟。
- **💾 上下文前缀缓存** - 当长会话超出 32k tokens 时，自动启用 Google 上下文前缀缓存，大幅降低 Token 计费成本并提升响应速度。

## 功能特性

- **智能代理与工具调用 (Agent mode)**：能够在您的库中执行主动任务的 AI 智能代理！它可以搜索文件、读取内容、创建新笔记、精确编辑现有笔记、重命名/移动文件、创建文件夹，甚至执行带有引用出处的深度研究。支持持久化会话、细粒度的工具确认机制，以及允许在写入前检查更改的 Diff 差异对比审查视图。
- **并发工具执行 [新]**：在多步骤代理运行期间并行执行独立的只读或搜索工具，以大幅降低延迟。同时状态修改类工具保持顺序执行，避免写入冲突。
- **上下文前缀缓存 [新]**：当会话大小超过 32k tokens 时，自动启用 Google 的上下文缓存来记住历史记录，极大减少长对话中 Token 的消耗量并提升首字响应速度。
- **文件 API 托管 [新]**：在会话中把音视频、PDF 等大文件仅上传一次至 Google 托管 API，后续对话均以轻量 URI 引用，避免反复发送 base64 负载导致的网络和流量开销。
- **本地语义搜索 (RAG)**：通过含义而非仅仅是关键字搜索您的库。使用 Google 文件搜索 API 在后台索引您的笔记。即使您不记得确切的字词，AI 也能找到相关内容。支持 PDF 和附件。
- **上下文感知代理**：将特定笔记添加为代理会话的持久上下文。代理可以在整个对话过程中访问和引用这些上下文文件，提供高度个性化的回答。
- **智能文档摘要**：一键生成笔记的简明单句摘要，并自动写入文档的 YAML frontmatter（默认键为 `summary`）。
- **选区 AI 功能**：右键选中文本可执行以下功能：重写 (Rewrite)、解释选区 (Explain Selection) 以及针对选区提问。
- **IDE 级文本自动补全**：在您输入时获得实时的、上下文感知的文本补全。使用 `Tab` 键接受补全。
- **持久化代理会话**：代理的对话历史记录将以普通 Markdown 文件的形式保存在您的库中（`gemini-scribe/Agent-Sessions/`），方便您进行备份、同步及历史回顾。
- **自定义提示词系统**：为代理会话创建可重用的系统提示词模板，定制特定场景的工作流（如代码开发、学术整理等）。
- **图片粘贴与多模态分析**：直接将截图粘贴到聊天输入框，上传至 Gemini 进行图像分析，同时文件会保存到您的库中并可以链接嵌入。
- **MCP 服务支持**：支持接入 MCP (Model Context Protocol) 服务，通过 stdio 管道或 HTTP 传输协议接入外部丰富的扩展工具包。
- **自动化计划任务 (Scheduled Tasks)**：在 Obsidian 后台根据 Cron 表达式或时间周期自动静默执行定时任务（如每日总结、周报、库维护），生成的文件直接写入库中。
- **库事件生命周期钩子 (Lifecycle Hooks)**：根据文件的创建、修改、删除和重命名等生命周期事件自动触发 AI 代理静默执行。
- **项目隔离 (Projects)**：针对库的不同目录划分子项目配置文件，隔离自定义指令、可用工具和可用技能集。
- **代理技能系统 (Agent Skills)**：遵循 agentskills.io 规范，支持加载和编辑外部专业技能包扩展代理的业务知识。
- **数据隐私保障**：与 Gemini API 的所有交互都直接在您的本地机器发起。除 Google 外，数据绝不发送给任何第三方服务器。历史记录完全保存在本地库中。

## 快速上手

1. 在社区插件市场搜索并安装 `Gemini Scribe`。
2. 前往 [Google AI Studio](https://aistudio.google.com/apikey) 获取免费的 API Key。
3. 在 Obsidian 设置 -> 社区插件 -> Gemini Scribe 中粘贴 API Key。
4. 点击侧边栏功能图标或运行命令面板指令“Open Gemini Chat”即可开启 AI 代理面板！
